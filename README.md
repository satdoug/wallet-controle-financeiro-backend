# Controle Financeiro Pessoal

Para criação desta aplicação serão utilizados no Backend Java com Spring Boot e no Frontend React.

## Estrutura Base do Projeto


## Pessoa
> - id
> - nome    
> - dataAniversario
> - salario
> - lista carteiras

## Usuario extends Pessoa
> - login
> - senha

## Carteira
> - id
> - descricaoCarteira
> - list<Transacoes> entradas    
> - list<Transacoes> saidas
    
## Transacoes
> - id
> - dataTransacao
> - descricao
> - tipoTransacao
